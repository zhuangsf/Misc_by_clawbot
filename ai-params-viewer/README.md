# 🦞 AI参数查看器 v0.3.0

一个极简的网页界面，用于查看小龙虾AI的核心配置文件和记忆记录。

## 🚀 功能特点

- **极简设计** - 只做必要的功能
- **零配置** - 开箱即用
- **快速访问** - 30秒内看到内容
- **安全只读** - 不修改任何文件
- **响应式** - 支持手机和电脑
- **智能布局** - 配置文件与记忆文件分离显示
- **扩展友好** - 支持大量记忆文件

## 📁 查看的文件

### 核心配置文件（标签式导航）
1. **SOUL.md** - 小龙虾的灵魂文件（你是谁）
2. **USER.md** - 用户配置文件（你帮助谁）
3. **AGENTS.md** - 代理工作方式（你的工作方式）
4. **IDENTITY.md** - 身份信息（你的基本信息）
5. **TOOLS.md** - 工具配置（你的工具配置）
6. **HEARTBEAT.md** - 心跳检查清单（你的检查清单）

### 记忆文件（下拉选择器）
- **智能发现** - 自动扫描 `memory/` 目录
- **日期排序** - 最新的记忆文件在最前面
- **优雅显示** - 下拉选择器，支持大量文件
- **特殊渲染** - 记忆文件有独特的紫色主题
- **格式：** `YYYY-MM-DD.md`

## 🛠️ 使用方法

### ⚠️ 重要：启动目录
由于文件位于上级目录，**必须在workspace根目录启动HTTP服务器**：

### 方法1：Python HTTP服务器（推荐）
```bash
# 关键：在workspace根目录启动
cd /root/.openclaw/workspace
python3 -m http.server 8080
```
然后在浏览器访问：`http://localhost:8080/ai-params-viewer/`

### 方法2：在ai-params-viewer目录启动
如果要在子目录启动，需要确保文件可访问：
```bash
cd /root/.openclaw/workspace/ai-params-viewer
# 创建符号链接到上级目录的文件
ln -sf ../*.md .
python3 -m http.server 8080
```
访问：`http://localhost:8080`

### 方法3：Node.js serve
```bash
cd /root/.openclaw/workspace
npx serve
```
访问：`http://localhost:3000/ai-params-viewer/`

### 方法4：直接打开文件
在支持CORS的浏览器中，可以直接打开`index.html`文件（可能需要调整浏览器安全设置）。

## 🔒 安全性

- **只读访问**：无法修改任何文件
- **白名单**：只能访问指定的文件
- **本地运行**：默认只在本地网络可访问
- **无认证**：简化设计，适合内部使用

## 🚧 限制

1. **只读**：无法编辑文件内容
2. **大文件**：大文件可能加载较慢
3. **CORS**：可能需要HTTP服务器
4. **权限**：需要文件读取权限

## 🦞 开发理念

这个项目体现了**第一性原理思维**：

1. **问题**：需要快速查看配置文件
2. **现状**：文件已存在，浏览器能显示文本
3. **方案**：用最少代码连接两者

**如果这个方案太复杂，说明我想错了。**
