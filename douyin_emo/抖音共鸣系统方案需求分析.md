## 🔥 第一性原理拆解

### 你的用户真相

| 维度 | 真相 |
|------|------|
| 用户是谁 | 18-25岁EMO女生，夜晚独行者 |
| 核心痛点 | "没人懂我" — 这是孤独感，不是"为情所困" |
| 底层需求 | 被理解 → 被看见 → 找到同类 |
| 你提供的价值 | 替她说出台词 — 她脑子里想了100遍但没说出口的话 |

### 核心洞察

抖音的本质不是"内容消费"，是"情绪代偿"。

你的用户不需要"解决方案"，她们需要的是：
有人替我说出我说不出口的话
有人懂我的感受，哪怕只是几秒钟
我不是一个人

## 🎯 情感触发器矩阵（底层逻辑）

不是"情感/段子分类"，而是4种钩子：

| 钩子类型 | 触发机制 | 例子 |
|----------|----------|------|
| 替我说 | 说出用户心里话 | "有些话你从来不说，但有人懂" |
| 与我同频 | 描述她的状态 | "凌晨三点，你也在等一个人消息吗" |
| 扎心真相 | 反常识洞察 | "其实不是没人追你，是你不想要" |
| 留白召唤 | 不说全，让她补充 | "那句话你们都懂，就是不敢说" |

15秒视频公式：
0-3s：替我说（Hook）
3-8s：与我同频（共鸣）
8-12s：扎心真相（情绪峰值）
12-15s：留白召唤（评论区引流）

class HookEngine:
    HOOK_TYPES = {
        "speak_for_me": {
            "template": ["有些话你从来不说，但...", "其实你只是想..."],
            "pattern": r"(替|帮|代替).*说",
        },
        "resonate_with_me": {
            "template": ["凌晨三点，你也在...", "其实你只是..."],
            "pattern": r"(你|我|我们).*(也|都).*在",
        },
        "harsh_truths": {
            "template": ["不是...而是...", "其实你心里清楚..."],
            "pattern": r"(不是|其实|说白了).*(是|就)",
        },
        "open_endings": {
            "template": ["那句话你们都懂...", "...就不用我说了吧"],
            "pattern": r".*（留白）.*",
        }
    }
