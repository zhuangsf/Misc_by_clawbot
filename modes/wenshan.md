## 核心风格

### 四大基石

#### 1. 一语双关
- 字面意思与深层含义并存
- 表层叙事 + 深层隐喻
- 仁者见仁智者见智

#### 2. 留白艺术
- 不说完，让读者自己补全
- 空白处都是戏
- 言有尽而意无穷

#### 3. 禅意哲思
- 看似说物，实则说人
- 物我合一，天人合一
- 东方哲学 + 现代情感

#### 4. 意象叠加
- 用具体物象承载抽象情感
- 画面感强，可视化
- 短句为主，干净利落

## 创作手法（七步成诗）

### 1. 悖论开场
用看似矛盾的话引发思考
```
例："你最懂你的人，往往最不懂你"
例："最亲近的人，最遥远的距离"
```

### 2. 以物喻人
借景抒情，物我合一
```
例："叶子的离开，是风的追求，还是树的不挽留"
例："下雨了，有人撑伞，有人淋雨"
```

### 3. 时空交错
过去 + 现在 + 未来，制造时光感
```
例："三十岁的你，活成了当时想要的样子吗"
例："十年前的梦想，现在还是梦想吗"
```

### 4. 疑问钩子
结尾留问题，不给答案
```
例："然后呢？然后你就真的自由了吗？"
例："所以呢？所以你决定了吗？"
```

### 5. 否定哲学
用"不是"来定义"是"
```
例："这不是结束，是另一种开始"
例："不是孤独，是独处"
```

### 6. 数字玄机
用数字制造思考空间
```
例："第1001次失眠，和第1次有什么不同"
例："三年了，你变了多少？"
```

### 7. 降维打击
用宏大视角看待微小事物
```
例："星星之火，可以燎原——也可以随风熄灭"
例："一个人的一生，在宇宙里是一秒"
```

## 输出标准

### 格式要求
- 短句为主，干净利落
- 画面感强，可拍成视频
- 每句都能单独成篇
- 适合15-60秒视频配音

### 情感共鸣
- 引发"被读懂"的感觉
- 替她说出台词
- 让她觉得"终于有人说出来了"
